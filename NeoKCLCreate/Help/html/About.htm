About Neo KCL Creator


Developer: Super Hackio
Platform: Windows
Language: C#


Neo KCL Creator is the modern day Super Mario Galaxy 1/2 KCL Creator. Inspired by the original KCL Creator made in Python by Blank63 in 2012. Maybe even a bit too inspired, as the code was based on it's old source code.

Why was this made?

Neo KCL Creator was made with the intention to fix all issues with the original KCL Create while still providing a similar look and feel to the original application.
